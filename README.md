**Инструкция по запуску проекта:**
1. Сохранить репозиторий локально.
2. Установить необходимые пакеты:
    в основной директории** Shop.Project**, внутренних директориях **Shop.Admin** и **Shop.API** использовать **npm install**;
    в директории **Shop.Client** использовать **yarn install**.
3. **Собрать клиентское приложение** - в директории Shop.Clietn выполнить команду **yarn build**
4. В основной директории Shop.Project выполнить команду **nodemon**.
